Focus Service Examples

This page demonstrates comprehensive usage of the FocusService for managing focus in Angular applications.

1. Dialog Focus Management

When opening a dialog, focus should move to the dialog and restore when closed.

@if (showDialog()) {

Example Dialog

}

2. Form Validation & Error Focus

Invalid form fields are automatically focused when submission fails.

@if (showForm()) {
@if (demoForm.get('name')?.invalid && demoForm.get('name')?.touched) { Name is required }
@if (demoForm.get('email')?.invalid && demoForm.get('email')?.touched) { Valid email is required }
@if (demoForm.get('phone')?.invalid && demoForm.get('phone')?.touched) { Phone is required }
}

5. Focus Origin Information

The FocusService uses Angular CDK's FocusMonitor to track how focus was applied:

  • keyboard: Focus via keyboard (Tab key)
  • mouse: Focus via mouse click
  • program: Focus set programmatically
